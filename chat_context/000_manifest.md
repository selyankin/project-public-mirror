# Manifest папки chat_context

Эта папка содержит контекст проекта для snapshot CI и анализа Максосом.  
Каждый файл имеет свою функцию:

- 001_project_overview.md — общий обзор проекта
- 002_architecture.md — архитектура и слои проекта
- 003_current_tasks.md — список задач и их статусы (отмеченные галкой - выполненные)
- 004_decisions_log.md — принятые решения и rationale
- 005_backlog.md — бэклог будущих задач и идей
- 006_tech_stack.md — используемые технологии и стек
