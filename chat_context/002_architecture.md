# Architecture

## Основные слои
1. **Data Layer**
   - Источники: судебные базы, жалобы, реестры, сайты объявлений
   - Хранение: PostgreSQL / MongoDB
2. **Backend Layer**
   - Python FastAPI
   - Парсинг, агрегирование, обработка данных
3. **Use Case Layer**
   - Генерация отчетов по пользователю / застройщику
   - Логика проверки «чистоты» продавца
4. **CI/CD & Snapshot**
   - Папка chat_context хранит snapshot проекта для анализа Максосом
