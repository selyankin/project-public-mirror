# Project Overview

## Название проекта
KQ Realty

## Цель проекта
Создать сервис, который собирает всю информацию о застройщиках и физических лицах, продающих недвижимость, включая судебные разбирательства, жалобы, рейтинги, для анализа и продажи отчетов.

## Основная идея
- Публичные зеркала и отчеты для физических лиц
- Отчеты о застройщиках
- Монетизация через продажу аналитических отчетов
- Прозрачность и предотвращение рисков для покупателей

## Целевая аудитория
- Люди, покупающие недвижимость
- Инвесторы и риелторы
- Проверяющие агентства

## Основные pain points
- Недобросовестные застройщики
- Судебные споры и блокировка квартир
- Отсутствие прозрачной информации о продавцах вторички
